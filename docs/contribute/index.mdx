# Contributing to CIROH Hub

Thank you for your interest in contributing to CIROH's Hub, our technical documentation site! Whether you're commenting on project information or submitting a pull request, we appreciate all kinds of contributions. This guide will help you understand the process of contributing code to the CIROH Hub.

Our website is constructed using [Docusaurus](https://docusaurus.io/), a modern static website generator.
